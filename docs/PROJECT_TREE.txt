==============================================
核心组件说明
==============================================

1. ScheduledBuildTask.java (124行)
   - 预约任务的数据模型
   - 包含ID、任务名、时间、参数等属性
   - 提供状态判断和转换方法

2. ScheduledBuildManager.java (245行)
   - 全局单例管理器
   - 管理所有预约任务
   - 负责调度和执行
   - 持久化存储

3. ScheduledBuildProperty.java (28行)
   - Jenkins任务属性扩展
   - 启用预约构建功能开关
   - 在任务配置页面显示

4. ScheduledBuildAction.java (215行)
   - Web界面控制器
   - 处理HTTP请求
   - 展示预约列表
   - 执行添加/取消操作

==============================================
构建产物
==============================================

构建命令:
  ./build.sh
  或
  mvn clean package

生成文件:
  target/scheduled-build.hpi  (Jenkins插件文件)

安装方法:
  1. Jenkins管理界面上传
  2. 复制到 $JENKINS_HOME/plugins/
  3. mvn hpi:run (开发模式)

==============================================
快速开始
==============================================

1. 构建插件:
   ./build.sh

2. 安装到Jenkins:
   上传 target/scheduled-build.hpi

3. 启用功能:
   任务配置 → 勾选"启用预约构建"

4. 添加预约:
   任务页面 → "预约构建" → 填写表单

详细说明请参考 QUICKSTART.md
